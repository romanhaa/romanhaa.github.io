Creating maps programmatically

Project status: ongoing
Last update: Jun 24, 2022
Code: romanhaa/plot-maps

In 2019, I wanted to create a map of a small Italian village as a gift for someone special. Once I started getting the hang of it, I made more. The linked repository contains all maps I made (at least the ones I finished), including instructions how to get the data and how to turn the data into the shown map.

So far, the maps are exclusively created with R.
